핵심 설계 원칙

CLI = 무료 우선
Codex/Claude/Gemini CLI → Google Translate 폴백. API 키 없이 사용 가능
Backend = Protocol
TranslationBackend 프로토콜 구현만으로 새 백엔드 추가. 파일 1개
pip install 단계별
기본: CLI만 / [ocr]: Surya+Tess / [web]: FastAPI+React / [all]: 전체
레이아웃 = 최우선
Redaction API + insert_htmlbox + 폰트/색상 보존. 현재 대비 대폭 개선
