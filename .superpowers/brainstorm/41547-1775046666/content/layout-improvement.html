구현 상세

Step 1: Redact
page.add_redact_annot(rect)
페이지 전체를 모아서
page.apply_redactions(
images=0, graphics=0)
→ 텍스트만 삭제
Step 2: Measure
font = fitz.Font("cjk")
width = font.text_length(
text, fontsize)
→ 정확한 글리프 측정
→ rect에 맞게 fontsize 조정
Step 3: Insert
page.insert_htmlbox(rect,
html_with_css,
scale_low=0)
→ 색상/폰트/정렬 CSS로
→ 자동 줄바꿈 + 축소

추가 개선: 레이아웃 인식

2단 레이아웃 감지
Elements의 bbox x좌표 클러스터링으로
단 구분 자동 감지. 왼쪽 단 → 오른쪽 단
순서로 번역하여 읽기 순서 보존
텍스트 확장 처리
EN→KO 번역 시 텍스트가 길어지는 경우
1) fontsize 축소 (최소 4pt까지)
2) insert_htmlbox scale_low=0 자동 축소
3) 그래도 넘치면 ellipsis + 풀텍스트는 MD로
